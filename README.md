# My Portfolio Website

Welcome to my personal portfolio! 🎉  
This project showcases my skills, projects, and career objectives as a final-year MIS student preparing for DevOps, Sysadmin, and QA/Testing roles.

---

## 🚀 Features

- **Terminal-inspired UI** with dark theme and green text.
- Sections for:
  - Career Objective
  - Education
  - Strengths
  - Skills
  - Projects
  - Contact Information
- Responsive design for desktop and mobile.
- Integrated CV download button.

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS (custom styling with terminal theme)
- **Backend/Hosting:** Nginx on Arch Linux
- **Version Control:** Git & GitHub
- **Dynamic DNS:** No-IP client for public access
